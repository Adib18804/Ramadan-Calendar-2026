# Ramadan-Calendar-2026
**রমজান ক্যালেন্ডার ১৪৪৭** ঢাকা-ভিত্তিক একটি ডিজিটাল প্ল্যাটফর্ম। ইসলামিক ফাউন্ডেশনের তথ্যে তৈরি এই সাইটে আছে সাহরি ও ইফতারের লাইভ কাউন্টডাউন, রঙিন প্রগ্রেস বার এবং ৩০ দিনের পূর্ণাঙ্গ সময়সূচী। আশরা ভিত্তিক ফিল্টার ও লাইলাতুল কদর চিহ্নিত করার সুবিধা সম্বলিত এই সাইটটি আধুনিক ডিজাইন ও বাংলা সংখ্যায় সাজানো, যা রমজানে ইবাদতকে সহজতর করবে।
