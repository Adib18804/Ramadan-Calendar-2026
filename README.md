# রমজান ক্যালেন্ডার ১৪৪৭ (Ramadan Calendar 1447)

## Overview
This is a web application for the Ramadan 1447 AH (2026 CE) calendar tailored for Dhaka, Bangladesh. It provides accurate Sehri, Iftar, and prayer times based on Islamic Foundation Bangladesh data. Features include live countdown timers, Hadith carousel, essential Duas, a Tasbih counter, theme toggle, print functionality, and WhatsApp sharing. The site is responsive, accessible, and designed with an Islamic aesthetic.

## Features
- **Live Timers**: Countdowns for Sehri and Iftar with progress bars and warnings.
- **Calendar Grid**: 30-day timetable with filters, expandable details, and Laylatul Qadr highlights.
- **Hadith Section**: Carousel of authentic Hadiths on Ramadan with auto-play.
- **Duas Section**: List of key supplications in Arabic, Bengali, and English.
- **Tasbih Counter**: Digital misbaha with rounds, undo, and reset.
- **Theme Toggle**: Light/dark mode with local storage persistence.
- **Sharing & Print**: WhatsApp share for daily timings; printable calendar.
- **Sticky Indicator**: Next prayer reminder.
- **Accessibility**: ARIA labels, keyboard navigation, and semantic HTML.

## Technologies Used
- HTML5, CSS3 (with CSS variables for theming).
- JavaScript (vanilla) for dynamic features like timers, carousels, and interactions.
- Fonts: Amiri (Arabic), Noto Sans Bengali, Playfair Display.
- No external libraries; pure vanilla JS for lightweight performance.

## Setup & Installation
1. Clone the repository: `git clone https://github.com/yourusername/ramadan-calendar-1447.git`
2. Open `index.html` in a modern browser (Chrome, Firefox, etc.).
3. No server required; runs locally as a static site.
4. For development, use a live server extension in VS Code for hot reloading.

## Data Source
- Timings from Islamic Foundation Bangladesh for Dhaka (23.7°N, 90.4°E).
- Hadiths and Duas sourced from authentic collections (Sahih Bukhari, Muslim, etc.).

## Customization
- Update `ramadanData` array in JS for different years/locations.
- Modify CSS variables in `:root` for color/theme changes.
